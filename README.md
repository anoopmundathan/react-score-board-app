## Getting Started
### Setup
```
$ git clone https://github.com/anoopmundathan/react-score-board.git
$ cd react-score-board
```
### Install
```
$ npm i
```
### Run
``` 
$ npm start
```
### View App
``` 
http://localhost:8000
```
